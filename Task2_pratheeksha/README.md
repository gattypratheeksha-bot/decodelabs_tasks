Task 2: IRIS FLOWER CLASSIFICATION USING DECISION TREE

I developed a Machine Learning model using Python to classify Iris flowers into different species. The model uses the Decision Tree algorithm and is trained on the Iris dataset. The dataset is divided into training and testing sets, and the model predicts the flower species based on input features such as sepal length, sepal width, petal length, and petal width.

Features

Loads the Iris dataset
Splits data into training and testing sets
Uses the Decision Tree Classification algorithm
Trains the model using training data
Predicts flower species on test data
Calculates and displays model accuracy
Demonstrates a basic machine learning workflow
