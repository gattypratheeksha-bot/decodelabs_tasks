Task 3: MOVIE RECOMMENDATION SYSTEM

I developed a simple Movie Recommendation System using Python. The program recommends movies based on the user's favorite category. It stores movie recommendations in a dictionary and displays a list of movies according to the category selected by the user.

Features

Displays available movie categories
Accepts user input for favorite movie category
Recommends movies based on the selected category
Uses a dictionary to store movie data
Handles invalid categories with an error message
Simple and user-friendly interface

Technologies Used

Python
VS Code
GitHub

Improved understanding of user input handling
Used conditional statements for decision-making
Implemented a basic recommendation system
Enhanced problem-solving and logical thinking skills
Gained experience in organizing and retrieving data efficiently
Understood the fundamentals of recommendation-based applications
