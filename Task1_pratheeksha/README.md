Task 1: RULE-BASED AI CHATBOT

I developed a simple Rule-Based AI Chatbot using Python. The chatbot interacts with users through text-based input and provides predefined responses based on specific keywords and questions. The chatbot runs continuously until the user enters "bye" to end the conversation.

Features

Greets users when they type "hello" or "hi"
Responds to "How are you?"
Introduces itself when asked "What is your name?"
Exits the chat when the user types "bye"
Handles unknown inputs with a default response
Runs continuously using a loop

Python
VS Code
GitHub
